# 企业信用信息自动查询工作流

这个工具用于批量打开国家企业信用信息公示系统查询企业主体信息，保存每家企业的 PDF，并导出汇总表。

## 能做什么

- 读取 `.xlsx` 或 `.csv` 公司名单
- 自动打开查询页面并逐家公司搜索
- 遇到登录、实名、人机验证、验证码时暂停，等你手动处理后继续
- 进入匹配的企业详情页
- 优先尝试使用页面上的报告/打印/下载入口保存 PDF
- 若官网没有直接下载按钮，则把当前详情页打印保存为 PDF
- 导出 `results.xlsx`，包含：
  - 公司全称
  - 住所地
  - 统一社会信用代码
  - 法定代表人姓名
  - 公开联系电话
  - 电话来源
  - PDF 文件路径
  - 查询状态
  - 失败原因

## 名单格式

Excel 或 CSV 至少包含一列：

- `公司名称`

也可以额外提供：

- `统一社会信用代码`

如果没有表头，建议按 `sample_companies.csv` 的格式整理。

## 第一次安装

在当前目录运行：

```bash
./company_credit_workflow/setup.sh
```

## 开始查询

```bash
./company_credit_workflow/run.sh 你的公司名单.xlsx
```

查询过程中会打开一个浏览器窗口。第一次使用请先登录/实名认证；遇到验证码、人机验证、企业选择不确定时，程序会提示你在浏览器里处理，回到终端按回车继续。

## 输出位置

默认输出到：

```text
company_credit_workflow/output/
  results.xlsx
  pdf/
  logs/
```

## 重要说明

这个工具只提取官网公开显示的信息。法定代表人手机号通常不是公开工商登记字段，工具不会绕过登录、验证码、人机验证，也不会从非授权来源抓取隐私手机号。电话字段会尽量从公开页面、年报或 PDF 文本中提取，并在结果表里标注来源。
